## Project Summary

The project implements a smart battery monitoring system using an ESP32 microcontroller to collect battery parameters and a Random Forest Regression model to predict the battery's State of Charge (SOC). The system provides real-time monitoring and accurate SOC estimation to improve battery safety and performance. :contentReference[oaicite:0]{index=0}
